```markdown
# Intervals AI Coach

Transform your Intervals.icu training data into professional coaching insights with AI-powered analysis.

## ✨ What It Does

Intervals AI Coach seamlessly integrates with your Intervals.icu activity pages, providing instant, intelligent analysis of your training sessions. Get personalized feedback that combines data-driven insights with the warmth of a professional coach.

## 🎯 Key Features

• **Instant AI Analysis** - One-click analysis of your workout data directly on Intervals.icu
• **Professional Coaching Insights** - Receive structured feedback on your training performance
• **Multi-Provider Support** - Works with OpenAI, DeepSeek, API2D, or any OpenAI-compatible API
• **Customizable Prompts** - Tailor the analysis style to your preferences
• **Privacy First** - All data processing happens through your own API key
• **Bilingual Support** - Automatic language detection (English/Chinese)

## 🚀 How to Use

1. Install the extension
2. Go to Settings and configure your AI API key (OpenAI, DeepSeek, or compatible services)
3. Navigate to any activity page on Intervals.icu
4. Click the floating "AI Coach" widget
5. Hit "Analyze" to get instant feedback

## 📊 What You'll Get

✅ **Direct Assessment** - A friendly one-sentence summary of your training
✅ **Key Observations** - 2-4 specific points with data-backed insights
✅ **Data Context** - Relevant metrics like cadence, heart rate zones, and pace
✅ **Actionable Suggestions** - Practical recommendations for improvement

## 🔒 Privacy & Security

- Your API key is stored securely in Chrome's sync storage
- No data is sent to third parties except your chosen AI provider
- You maintain full control over your data

## 💡 Perfect For

- Endurance athletes (runners, cyclists, swimmers, triathletes)
- Intervals.icu users seeking deeper training insights
- Athletes who want professional coaching feedback without the cost

## ⚙️ Requirements

- An Intervals.icu account
- An API key from OpenAI, DeepSeek, or compatible services
- Chrome browser (version 88+)

## 🆘 Support

If you encounter issues or have suggestions, please visit our GitHub repository or contact us through the support link.

---

**Note:** This extension requires an API key from a supported AI service provider. API usage may incur costs from your chosen provider.
```

## 六、注意事项

1. **审核时间:** Chrome Web Store 审核通常需要 1-3 个工作日
2. **可能被要求提供隐私政策:** 如果审核团队要求，可以使用 GitHub Pages 快速创建
3. **manifest.json 中的 host_permissions:** 当前使用 `*://*/*` 可能被审核团队质疑，建议只保留必需的域名：
   ```json
   "host_permissions": [
     "https://intervals.icu/*",
     "https://api.openai.com/*",
     "https://api.deepseek.com/*",
     "https://openai.api2d.net/*"
   ]
   ```
4. **首次发布费用:** Chrome Web Store 开发者注册需一次性支付 $5 USD
