心脏恢复四象限图（示例）

说明（实现细节）
  • 横轴 x = ΔHRV = HRV_7day_avg - HRV_60day_baseline （单位：lnRMSSD）
  • 纵轴 y = ΔRHR_good = RHR_60day_baseline - RHR_7day_avg （单位：bpm；正值表示夜间RHR下降 / 情况变好）
  • SWC（smallest worthwhile change）以绝对值形式在两轴画出 ±SWC 窗口（示例值：HRV_SW C = 0.10 lnRMSSD, RHR_SWC = 1.5 bpm）。
  • 象限定义（对应文中 4 状态）：
    1. 状态1（良好适应）：ΔHRV ≥ -SWC && ΔRHR_good ≥ -SWC （HRV ↑ 或在SWC内，RHR ↓ 或在SWC内）→ 绿灯
    2. 状态2（急性疲劳）：ΔHRV < -SWC && ΔRHR_good < -SWC （HRV ↓ 且 RHR ↑）→ 红灯
    3. 状态3（累积疲劳 / 高HRV 高HR悖论）：ΔHRV > +SWC && ΔRHR_good < -SWC（HRV ↑ 且夜间RHR ↑）→ 黄/红
    4. 状态4（慢性疲劳）：ΔHRV < -SWC && ΔRHR_good > +SWC（HRV ↓ 且 RHR ↓ —— 注：这里用 ΔRHR_good 表示基线 - 当前，若为负则表示RHR上升）
恢复分数（0–100）—— 公式概述
Score = clamp( 50 + 25 * z_HRV + 25 * z_RHR , 0,100 ) + stateAdjustment

其中：

  • z_HRV = (HRV_7 - HRV_60) / SWC_HRV（在 -3..+3 之间截断）
  • z_RHR = (RHR_60 - RHR_7) / SWC_RHR（正表示更好；在 -3..+3 截断）
  • stateAdjustment：如果处于高风险状态会减分（状态2：-30，状态3：-20，状态4：-35；状态1：0）

提示：参数（SWC、权重、状态惩罚）均可按目标人群或经验数据微调。
