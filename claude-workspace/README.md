# Claude Code 工作区

这是专门用于 Claude Code 开发和实验的工作目录。

## 目录说明

- **experiments/** - 实验性代码和新技术尝试
- **learning/** - 学习新技术、框架、语言的练习代码
- **sandbox/** - 临时测试和沙盒环境

## 使用建议

1. 每个子项目建议创建独立的文件夹
2. 临时测试放在 sandbox，测试完可以随时清理
3. 有价值的实验可以从 experiments 提升到外层作为正式项目
4. 学习笔记建议放在 ../Notes/ 目录

## 注意事项

- 这个目录下的代码会被外层的 Git 仓库管理
- 敏感信息请使用 .gitignore 排除
- 大文件和数据集建议放在其他位置
