## 关键术语定义

- **E / M / T / I / R**：丹尼尔斯跑步训练类型，包括 E（轻松跑）、M（马拉松配速）、T（乳酸阈值）、I（最大摄氧量）、R（重复跑）。强度递增关系：E < M < T < I < R
- **波动周期**：训练强度的交替模式，避免连续高强度训练，实现刺激-恢复-适应的循环
- **恢复训练模式**：中断训练后（3天以上且准备度<80%）的恢复性训练，优先低强度，仅持续一次
- **基础训练量**：根据用户等级和训练类型预设的标准训练量（如距离、时间、组数）
- **训练准备度**：综合评估的当日身体状态，范围 0-100%，数值越高代表状态越好

## 输入

- 用户可用训练类型集合：根据用户等级确定
- 当前目标之前的课程安排情况：历史训练记录
- 训练中断天数：距上次训练天数
- 训练准备度：0-100%，可能不存在
- 当前小周期内（7天）已完成的训练类型：本周已练过的类型
- 各训练类型的基础训练量：预设标准值

## 核心逻辑

```
开始
  │
  ├─ 是否是训练周期第一周？
  │   │
  │   ├─ 是 → 按预排的固定模式初始化课程
  │   │   例子：用户为进阶跑者，可用 {E, M, T, I, R}
  │   │   → 初始化排列为 E、I、M、T、R
  │   │
  │   └─ 否 → 继续
  │
  ├─ 计算中断天数（今日 - 上一次执行当前训练目标的日期）
  │   │
  │   ├─ 中断天数 ≤ 2天 → 正常波动逻辑
  │   │   例子：昨天练了 T，前天练了 E
  │   │   → E(低) - T(高) = 低高模式 → 今天选高，即 R 或 I
  │   │
  │   ├─ 中断天数 = 3天 → 判断训练准备度
  │   │   ├─ 准备度 ∈ (0, 80%) → 恢复训练模式
  │   │   ├─ 准备度 ≥ 80% → 正常波动逻辑
  │   │   └─ 准备度不存在 → 恢复训练模式
  │   │
  │   └─ 中断天数 ≥ 4天 → 恢复训练模式
  │
  ├─ 正常波动逻辑
  │   │
  │   ├─ 排除休息日，取前两次训练
  │   ├─ 判断模式：
  │   │   ├─ 低-高 → 这次选高
  │   │   ├─ 高-低 → 这次选低
  │   │   └─ 相同强度 → 随机选择
  │   │
  │   └─ 筛选训练类型：
  │       1. 排除本周已完成的类型
  │       2. 按强度要求筛选（高/低）
  │       3. 从结果中选择
  │
  ├─ 恢复训练模式
  │   │
  │   ├─ 优先从可用类型中筛选低强度训练（E 或 M）
  │   ├─ 如果本周没有 E 或 M：
  │   │   → 把本周的 E 课程复制一次到现在
  │   │   → 如果本周没有 E 课程，则根据用户训练水平生成一个 M 课程
  │   │
  │   └─ 下次训练恢复正常波动逻辑
  │
  └─ 训练量调整
      │
      ├─ 准备度 ≥ 80%：
      │   → 训练量 = 基础训练量 * (训练准备度 + 20%)
      │
      │   例子：基础训练量 10km
      │   - 准备度 85% → 10 * (85% + 20%) = 10 * 105% = 10.5km
      │   - 准备度 95% → 10 * (95% + 20%) = 10 * 115% = 11.5km
      │   - 准备度 100% → 10 * (100% + 20%) = 10 * 120% = 12km
      │
      ├─ 准备度 ∈ (60%, 80%)：
      │   → 训练量 = 基础训练量
      │
      │   例子：基础训练量 10km，准备度 70%
      │   → 训练量 = 10km
      │
      └─ 准备度 ∈ [40%, 60%)：
          → 训练量 = 基础训练量 * (训练准备度 + 40%)

          例子：基础训练量 10km
          - 准备度 55% → 10 * (55% + 40%) = 10 * 95% = 9.5km
          - 准备度 45% → 10 * (45% + 40%) = 10 * 85% = 8.5km
          - 准备度 40% → 10 * (40% + 40%) = 10 * 80% = 8km
```

## 局限性以及待优化点

- **初始化模式**：当前为预排固定序列，后续需考虑与波动周期的兼容性
- **恢复训练模式**：当前仅持续一次，后续可根据身体状态动态调整次数
- **训练量调整公式**：当前为线性关系，后续可考虑根据准备度区间使用不同的曲线函数
- **边界衔接**：80% 和 60% 两个临界点需要确保平滑衔接
- **用户自选类型**：当前待定义，后续需根据目标完成度、周期进展、训练准备度综合判断
- **配比平衡**：当前暂不处理，后续可加入动态权重修正
- **周期结束后安排**：当前为提醒休息，后续可根据目标完成度、身体状态安排
- **个体差异**：当前为通用公式，后续可根据用户历史数据个性化调整系数区间
- **训练类型差异**：当前未区分不同训练类型的敏感度（如 R 课可能对准备度更敏感）
