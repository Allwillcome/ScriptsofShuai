# 주관적 운동 강도(RPE)란?

주관적 운동 강도(RPE, Rate of Perceived Exertion)는 운동 강도를 주관적으로 평가하는 지표로, 이번 훈련에 대한 주관적인 노력 강도를 나타냅니다. 간단히 말하면 운동할 때 "얼마나 피곤한가"입니다. 심박수나 파워 같은 객관적 데이터와 달리, RPE는 종합적인 느낌——근육 피로, 호흡 곤란, 심리적 느낌 등 여러 차원의 경험을 반영합니다.

## 왜 RPE가 중요한가?

RPE의 핵심 가치는 "주관적 느낌"과 "객관적 데이터"를 연결한다는 것입니다. 연구에 따르면 주관적 피로감과 생리적 부담 사이에 강한 상관관계가 있습니다(Borg, 1998). 지구력 운동의 경우, RPE는 심박 스트랩이나 파워미터가 없어도 현재 훈련 강도가 목표 구간 내에 있는지 실시간으로 판단하는 데 도움이 됩니다.

또한, RPE는 "숨겨진 피로"를 식별하는 데도 도움이 됩니다. 몸이 한계에 달했는데 심박수가 아직 완전히 반영되지 않았을 수 있습니다. 이 경우 RPE는 추가적인 참고 지표를 제공하여 과도한 훈련을 피할 수 있게 합니다.

## RPE 평가 기준

RPE는 1~10 등급 평가 체계를 사용하며, 수치가 높을수록 피로도가 높습니다:

| RPE | 설명 |
|-----|------|
| 1 | 거의 노력 없음 |
| 2 | 매우 가벼움 |
| 3 | 좀처럼 가벼움 |
| 4 | 가벼움 |
| 5 | 중간 |
| 6 | 피로해짐 |
| 7 |Challenging |
| 8 | Hard |
| 9 | 매우 Hard |
| 10 | 극도로 Hard |

PeakWatch에서 사용자가 입력하거나 Apple Health 데이터가 없는 경우, 시스템은 운동 시간과 평균 심박수를 기반으로 RPE를 자동으로 계산하여 모든 훈련 기록에 참조 값을 제공합니다.

## PeakWatch의 RPE

PeakWatch는 세 가지 RPE 데이터 소스를 지원합니다:

1. **Apple Health 읽기**: PeakWatch는 Apple Health에 기록된 RPE 데이터를 읽을 수 있습니다(Apple Health에서 PeakWatch의 접근을 승인해야 합니다)
2. **수동 입력**: 운동 후 PeakWatch에서 주관적인 운동 강도를 수동으로 입력할 수 있습니다
3. **자동 계산**: 사용자가 입력하거나 Apple Health 데이터가 없는 경우, PeakWatch는 운동 시간과 심박수 데이터를 기반으로 RPE를 추정합니다

훈련 기록 상세 페이지에서 RPE는 "훈련 노력값" 위치 아래에 표시됩니다. RPE를 입력했거나 Apple Health에 관련 데이터가 있으면 시스템이これらの 값을 우선적으로 표시하고, 그렇지 않으면 자동 계산 결과를 표시합니다. RPE 옆의 코멘트는 수치에 따라 해당 설명 텍스트와 자동으로 매칭됩니다.

## 데이터 동기화 및 수정

PeakWatch가 운동을 시작하면 운동 데이터에 따라 RPE를 자동으로 계산합니다. 훈련 기록에서 RPE 값을 수정할 수 있으며, 수정하면 Apple Health에 동기화됩니다. 이는 RPE 데이터가 서로 다른 앱 간에 흐르면서 데이터 일관성을 유지할 수 있음을 의미합니다.
