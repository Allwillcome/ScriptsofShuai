# product-copywriting-opt

产品界面文案优化 Skill，遵循苹果 PACET 原则帮你写出更好的界面用语。

## 功能

- 优化现有界面文案
- 撰写新的提示文案/按钮文案/空状态
- 审核现有文案是否符合规范
- 提供具体修改案例

## 使用方式

当需要优化界面文案时，使用此 Skill：
- `/product-copywriting-opt`
- 或者在描述问题时带上"界面文案"、"文案优化"等关键词

## 核心原则

1. **Purpose** - 每个页面有明确目的
2. **Anticipation** - 想象对话，知道用户下一步
3. **Context** - 跳出 App 思考场景
4. **Empathy** - 为所有用户而写

## 必做检查

完成文案后必须**大声朗读**检查是否像对话、是否有错误。
